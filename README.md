# flog
Tiny Fortran module that provides colorable output;
Inpired by [flogging](https://github.com/DaanVanVugt/flogging);

# Build
```
$ git clone https://github.com/r-novel/flog.git && cd flog
```

```
$ make build
```

# Test
```
$ ./bin/flog
```
